# Number Guessing Game

A simple Python number guessing game built to practice Python fundamentals such as:
- loops
- functions
- conditionals
- modules
- user input handling
- exception handling

## Features

- Random number generation
- Limited attempts system
- Hint system (higher/lower)
- Modular code structure
- Input validation
